ZIP File

For now, you can download the release file using the below link. To install using this ZIP file, you will need:

  1. Extract the ZIP file into a folder
  2. Navigate to chrome://extensions in the browser search box
  3. Turn "Developer Mode" on
  4. Select "Load Unpacked"
  5. Select the folder where the extension files have been extracted to
  6. The extension should now be installed!
