Porkbun Search Filters

The search filters you wish Porkbun had with one extension!

Welcome to Porkbun Search Filters!

Hi! I'm Eddie and I use Porkbun to buy a lot of the website domains that I use for various web projects. I tend to find myself using Ctrl+F to find domains of a particular price and was getting so fed up of doing it, along with sifting through Handshake domains I have no care for, that I asked Porkbun over Twitter to add some filters and 'sort by' options to their search — they never answered. So I did it myself, in the form of a Chrome Extension and here it is for you to use!

Please read: Usage Guide

To use 'sort by' options, please wait until all domain prices have been loaded onto the search page. Only then, click the extension and change the 'sort by' option. The extension grabs the prices from the page when you click the extension, not automatically; please be patient, then click!

If the formatting and/or sorting does not work as intended, please make your search again and try applying the sorting option then
